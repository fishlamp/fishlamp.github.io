Welcome to the FishLamp homepage.

FishLamp is a collection of small collections of code I call pieces - these pieces are meant to be aggregated together into your project. Each piece is meant to solve a specific problem or implement a specific feature. FishLamp is not meant to be monolithic.

You consume and use these pieces using the awesome CococPods tools. (Note, I'm still working on wiring everything up to cocoapods, so we're not quite fully integrated yet).
